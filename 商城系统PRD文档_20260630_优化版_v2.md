# 商城系统产品需求文档(PRD) - 优化版 v2.0

## 文档信息
- **项目名称**: 全平台电商商城系统
- **文档版本**: v2.0 (优化版)
- **编写日期**: 2026-06-30
- **参考产品**: 云上聚乐商城 (https://ext.dcloud.net.cn/plugin?id=13788) + Shopro分销商城 (https://ext.dcloud.net.cn/plugin?id=1996)
- **目标平台**: 微信小程序、Android APP、iOS APP、Web管理端

---

## 一、项目概述

### 1.1 项目目标
开发一套完整的**分销商城系统**，支持多端部署（小程序、Android、iOS、Web管理端），具备商品管理、订单处理、会员体系、**分销体系**、**门店管理**、**多配送方式**等核心电商能力。

### 1.2 核心特色功能 (新增)
1. **分销系统** - 分销中心、分销关系、佣金结算
2. **门店系统** - 门店管理、到店自提、扫码核销
3. **多配送方式** - 到店自提、商家配送、自动发货
4. **发票管理** - 订单开票申请、发票记录
5. **售后/退款** - 完整售后流程、退款进度追踪
6. **钱包系统** - 余额管理、多方式提现
7. **积分系统** - 消费送积分、积分记录
8. **分享海报** - 分享链路追踪、海报生成

### 1.3 技术栈 (已确认)

#### 后端技术栈
- **框架**: Spring Boot 3.2.x
- **安全框架**: Spring Security + JWT
- **ORM框架**: MyBatis-Plus 3.5.x
- **数据库**: MySQL 8.0.46 (字符集 utf8mb4)
- **缓存**: Redis 7.x
- **接口文档**: Knife4j (Swagger增强)
- **工具库**: Hutool
- **文件上传**: 本地存储 / 对象存储 (腾讯云COS / 阿里云OSS)
- **支付对接**: 微信支付 SDK (含V3)、支付宝 SDK
- **物流对接**: 快递鸟 API
- **日志**: Logback
- **构建工具**: Maven 3.9+

#### 用户端技术栈 (多端复用)
- **跨平台框架**: uni-app (Vue3 + Vite)
- **状态管理**: Pinia
- **UI组件库**: uni-ui + uView UI
- **HTTP客户端**: uni.request 封装
- **分享海报**: canvas 绘图
- **多端编译**: 
  - 微信小程序 (HBuilderX编译)
  - Android APP (HBuilderX云打包/本地打包)
  - iOS APP (HBuilderX云打包/本地打包)
  - H5 (可选)

#### 管理端技术栈
- **框架**: React 18.x + TypeScript
- **UI组件库**: Ant Design Pro 5.x / Ant Design 5.x
- **状态管理**: Zustand / Redux Toolkit
- **路由**: React Router 6.x
- **HTTP客户端**: Axios
- **构建工具**: Vite 5.x
- **代码规范**: ESLint + Prettier
- **图表**: Ant Design Charts / ECharts
- **富文本编辑器**: WangEditor / TinyMCE

#### 部署架构
- **后端部署**: Docker + Docker Compose
- **前端部署**: Nginx (管理端静态资源)
- **反向代理**: Nginx
- **数据库部署**: Docker容器 / 独立服务器
- **HTTPS**: Let's Encrypt 免费证书 / 商业SSL证书
- **CI/CD**: Jenkins / GitLab CI (可选)

#### 第三方服务
- **支付**: 微信支付 (小程序/APP)
- **登录**: 微信登录 (小程序/APP)、短信验证码登录
- **物流**: 快递100 API / 快递鸟 API
- **推送**: UniPush (uni-app官方推送，支持Android/iOS)
- **存储**: 腾讯云COS / 阿里云OSS (商品图片/视频存储)
- **短信**: 阿里云短信 / 腾讯云短信
- **地图**: 腾讯地图 / 高德地图 (地址选择)
- **分享**: 微信分享、朋友圈分享、海报生成

## 三、功能模块详细需求 (优化版)

### 3.1 用户端功能模块

#### 3.1.1 首页模块
**功能描述**: 展示商城首页，包含轮播图、导航分类、推荐商品、秒杀商品、拼团商品、分销商品等。

**核心功能**:
1. **轮播图** - 支持跳转商品/活动页
2. **导航分类** - 图标+名称，支持自定义
3. **推荐商品** - 根据销量/评分推荐
4. **秒杀商品** (新增) - 限时秒杀，倒计时
5. **拼团商品** (新增) - 多人拼团，分享邀请
6. **分销商品** (新增) - 分销佣金展示
7. **优惠券领取** - 领券中心
8. **公告信息** - 滚动公告

**API接口**:
- `GET /api/home/banner` - 获取轮播图
- `GET /api/home/categories` - 获取导航分类
- `GET /api/home/recommend-products` - 获取推荐商品
- `GET /api/home/seckill-products` - 获取秒杀商品 (新增)
- `GET /api/home/group-products` - 获取拼团商品 (新增)
- `GET /api/home/distribution-products` - 获取分销商品 (新增)

#### 3.1.2 商品模块
**功能描述**: 商品列表、商品搜索、商品详情、商品评价、商品分享。

**核心功能**:
1. **商品列表** - 分类筛选、价格排序、销量排序
2. **商品搜索** - 关键词搜索、搜索历史
3. **商品详情** - 商品图片、价格、规格选择、详情介绍
4. **商品评价** - 评分、评价内容、评价图片
5. **商品分享** (新增) - 生成分享海报、分享到微信/朋友圈
6. **分销佣金展示** (新增) - 显示分销佣金金额

**API接口**:
- `GET /api/goods/list` - 商品列表
- `GET /api/goods/detail/{id}` - 商品详情
- `GET /api/goods/search` - 商品搜索
- `POST /api/goods/share-poster` - 生成分享海报 (新增)
- `GET /api/goods/distribution-commission/{id}` - 获取分销佣金 (新增)

#### 3.1.3 购物车模块
**功能描述**: 购物车列表、添加购物车、删除购物车商品、更新购物车数量。

**核心功能**:
1. **购物车列表** - 商品列表、合计金额
2. **添加购物车** - 选择规格、输入数量
3. **删除购物车商品** - 单删/批量删除
4. **更新购物车数量** - 增加/减少数量
5. **去结算** - 跳转到确认订单页

**API接口**:
- `GET /api/cart/list` - 购物车列表
- `POST /api/cart/add` - 添加购物车
- `DELETE /api/cart/delete/{id}` - 删除购物车商品
- `PUT /api/cart/update` - 更新购物车数量

#### 3.1.4 订单模块
**功能描述**: 确认订单、提交订单、订单列表、订单详情、订单取消、订单支付、订单物流、订单评价、退款/售后。

**核心功能**:
1. **确认订单** - 商品信息、收货地址、配送方式、优惠券、发票
2. **提交订单** - 创建订单、选择支付方式
3. **订单列表** - 全部/待付款/待发货/待收货/待评价
4. **订单详情** - 订单状态、商品信息、物流信息、发票信息
5. **订单取消** - 未付款订单可取消
6. **订单支付** - 微信支付/支付宝支付
7. **订单物流** - 物流追踪 (对接快递鸟)
8. **订单评价** - 评分、评价内容、上传图片
9. **退款/售后** (新增) - 申请退款、退款进度、售后记录
10. **多配送方式** (新增) - 到店自提/商家配送/自动发货

**API接口**:
- `POST /api/order/confirm` - 确认订单
- `POST /api/order/create` - 提交订单
- `GET /api/order/list` - 订单列表
- `GET /api/order/detail/{id}` - 订单详情
- `POST /api/order/cancel/{id}` - 取消订单
- `POST /api/order/pay/{id}` - 订单支付
- `GET /api/order/logistics/{id}` - 物流追踪
- `POST /api/order/review/{id}` - 订单评价
- `POST /api/order/refund/{id}` - 申请退款 (新增)
- `GET /api/order/refund-status/{id}` - 退款进度 (新增)
- `POST /api/order/invoice` - 申请发票 (新增)
- `GET /api/order/invoice-status/{id}` - 发票状态 (新增)

#### 3.1.5 支付模块
**功能描述**: 微信支付、支付宝支付、余额支付。

**核心功能**:
1. **微信支付** - 微信小程序支付、APP支付
2. **支付宝支付** - APP支付、H5支付
3. **余额支付** (新增) - 使用钱包余额支付
4. **支付结果回调** - 支付成功/失败处理

**API接口**:
- `POST /api/pay/wechat` - 微信支付
- `POST /api/pay/alipay` - 支付宝支付
- `POST /api/pay/balance` - 余额支付 (新增)
- `POST /api/pay/callback/wechat` - 微信支付回调
- `POST /api/pay/callback/alipay` - 支付宝支付回调

#### 3.1.6 用户模块
**功能描述**: 用户注册、用户登录、个人中心、收货地址、优惠券、收藏、浏览历史、会员中心。

**核心功能**:

- `GET /api/user/info` - 用户信息

- `GET /api/user/address` - 收货地址列表
- `POST /api/user/address` - 添加收货地址
- `PUT /api/user/address/{id}` - 编辑收货地址

- `GET /api/user/coupon` - 我的优惠券

- `GET /api/user/history` - 浏览历史

## 六、注意事项 (优化版)

### 6.1 小程序配置
1. 微信小程序需要配置服务器域名 (必须是HTTPS)
2. 微信小程序需要添加微信支付能力
3. 微信小程序需要配置消息推送 (模板消息)

### 6.2 APP配置
1. Android APP需要配置应用签名
2. iOS APP需要配置URL Scheme (微信登录/支付回调)
3. APP需要配置推送证书 (UniPush)

### 6.3 支付配置
1. 微信支付需要配置商户号、API密钥、证书
2. 支付宝支付需要配置应用ID、私钥、公钥
3. 支付回调需要配置服务器地址 (必须是公网可访问)

### 6.4 物流配置
1. 快递鸟需要配置用户ID、API密钥
2. 物流查询需要缓存物流信息 (避免频繁调用API)

### 6.5 分销配置 (新增)
1. 分销关系绑定需要设置有效期 (如：点击链接7天内下单有效)
2. 佣金结算需要设置结算周期 (如：订单完成后7天结算)
3. 提现需要设置最低提现金额、提现手续费

### 6.6 门店配置 (新增)
1. 门店地址需要配置经纬度 (用于计算距离)
2. 自提时间需要设置可选时间范围
3. 核销码需要设置有效期
